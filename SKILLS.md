# Design Repository

A local reference library for UI/UX design inspiration, organized by medium.

## Folder Structure

```
Design Repository/
├── SKILLS.md
├── Websites/
│   └── [Company Name]/
│       ├── Screenshots/   — Static UI captures (.png, .jpg)
│       ├── Video/         — Screen recordings of interactions (.mp4, .mov) — local only, not committed
│       └── notes.md       — Observations, tags, and reference links
├── Apps/
├── Print/
└── Social/
```

## How to Set Up a New Reference

1. Create a folder under the relevant category named after the company/project
2. Inside it, create `Screenshots/`, `Video/`, and `notes.md`
3. Fill in the `notes.md` using the template below
4. Drop your screenshots and screen recordings into the respective folders
5. Add a row to the **Current References** table at the bottom of this file

## Tagging Reference

Use these tags in `notes.md` files for easy searching:

**Layout & Structure**
`#layout` `#grid` `#hero` `#navigation` `#footer` `#sidebar` `#cards`

**Visual Style**
`#dark-mode` `#minimal` `#bold` `#editorial` `#glassmorphism` `#gradient`

**Typography & Color**
`#typography` `#color` `#monospace` `#serif` `#sans-serif`

**Motion & Interaction**
`#animation` `#scroll-effects` `#parallax` `#hover` `#transition` `#motion`

**Components**
`#forms` `#buttons` `#modals` `#tabs` `#carousel` `#accordion`

**Responsive**
`#responsive` `#mobile-first` `#accessibility`
